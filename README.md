## 광고 현황 대시보드 및 광고 관리 서비스

### 🚀 기능 요구사항
- 사이드바
  - 서비스 추가 버튼까지만, 실제 추가 기능은 구현 X
- 대시보드
  - 통합광고 현황까지만 구현
  - 매체 현황은 구현 X
- 광고관리
  - 광고 표시 및 수정하기까지만 구현 (백엔드가 없으므로 수정한 데이터는 새로고침 시 초기화되어도 무방)
  - 광고 만들기는 구현 X

- 어떤 상태로 다른 페이지에 이동했다 하더라도, 다시 기존 페이지로 복귀할 시 마지막 상태가 유지되어야 함 (필터링 상태 등)
- 임의적으로 Fetch 타임을 만들어 로딩 시간 지연
